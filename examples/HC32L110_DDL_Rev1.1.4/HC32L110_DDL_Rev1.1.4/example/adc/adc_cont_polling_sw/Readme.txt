功能描述
================================================================================
本样例主要展示ADC 单通道连续采样示例，采样结果累加十次，使用轮询方式获取ADC运行状态。
说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L110_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）查看串口输出



================================================================================
注意
================================================================================
1）打开工程编译并运行
2) ADC通道选择通道1，P26
3）在IAR Livewatch窗口（KEIL Watch窗口）查看转换结果变量u32AdcResultAcc的值
3) 配置ADC并开始采样，累加转换10次
   转换完成，读取累加转换结果，并将RESULT_ACC寄存器清零

================================================================================


================================================================================
注意
================================================================================
1）内置电压跟随器使能可以采样外部高阻信号，但是BUFEN使能后，最高SPS采样速率 <= 200K

2）参考电压:内部2.5V(Vcc > 2.8V,SPS <= 200kHz)  

3）参考电压:内部1.5V(Vcc > 1.8V,SPS <= 200kHz)  

4）SPS速率 = ADC时钟 / (采样时钟 + 16CLK) 

5）通道8（AIN8）不能用于连续采样通道
